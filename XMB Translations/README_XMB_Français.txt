1. Placez le fichier "psx-desr-2.10-trad-fr-v1.0.elf" sur un clé USB.
2. Insérez la clé USB dans la PSX et lancez wLaunchELF kHN.
3. Dans wLaunchELF, allez sur mass:/ et puis lancez le fichier "psx-desr-2.10-trad-fr-v1.0.elf".
4. Confirmez l'installation avec O.

Si vous voulez retourner à la langue Japonais, il suffit de lancer le fichier "psx-desr-2.10-trad-fr-v1.0.elf" encore une fois avec wLaunchELF.
