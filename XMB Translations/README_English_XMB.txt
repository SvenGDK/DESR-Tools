It is recommended to use u/wLaunchELF to do this process.

1. (Optional) Backup all .xml files on your PSX that will be replaced inside the folders (PhotoView.xml, barc.xml for example) to an external device
2. Copy the .xml files or the folder from the extracted archive inside the corresponding folder on the PSX (xosd) and replace the .xml file
3. Reboot the console
